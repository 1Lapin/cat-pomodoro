# 喵喵番茄钟 (Meow Pomodoro) 🐾

一个可爱猫咪主题的番茄钟网页应用，支持白噪音和猫咪奖励系统。

## 功能特点

- **经典番茄钟**：25分钟专注工作，结束后自动开启1分钟休息。
- **猫咪主题 UI**：纯 CSS 绘制的可爱猫咪，会根据工作/休息状态变换表情。
- **白噪音陪伴**：内置雨声、海浪声、森林声，支持音量调节。
- **小鱼干奖励系统**：每完成一个番茄钟获得一个小鱼干 🐟，并记录专注历史。
- **便捷控制**：支持空格键开始/暂停，R键重置。

## 如何使用

1. 下载或克隆仓库。
2. 在浏览器中打开 `index.html` 即可使用。
3. _建议通过本地服务器（如 VS Code 的 Live Server 扩展）打开，以获得最佳的音频和图片加载体验。_

## 技术栈

- HTML5
- CSS3 (Flexbox, Animations)
- JavaScript (ES6, Fetch API)
- [TheCatAPI](https://thecatapi.com/) - 奖励猫咪图片来源
